# Команды, чтобы  запустить програму:

1) git clone https://github.com/FflashkingF/Solitaire

2) cd Solitaire

3) mkdir build

4) cd build

6) cmake ..
   
7) make (после этой команды появился исполняемый файл - ../bin/Solitaire (Solitaire/bin/Solitaire)
